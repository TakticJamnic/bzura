# Bzura 1939
Bzura 1939 przedstawia bitwę która rozegrała się podczas Kampanii Wrześniowej 1939.

Gra udostępnia mapę i żetony, natomiast najlepiej jest grać korzystając z systemu WB-95 (Wielkie Bitwy 1939-1945) wydawnictwa Taktyka i Strategia. Instrukcję (albo całe gry) do systemu można zakupić m.in. tutaj:

* 👉 Sklep Taktyka i Strategia https://taktykaistrategiasklep.pl/
* 👉 Pudełkowa wersja Bzury 1939 https://taktykaistrategiasklep.pl/produkt/bzura-1939/
* 👉 System WB-95 Taktyka i Strategia https://taktykaistrategia.pl/kategoria-produktu/nasze-gry/?filter_system=wb-95&query_type_system=or

## 🔧 Jak uruchamiać

Do gry niezbędna jest aplikacja Vassal. Jest to darmowy silnik do uruchamiania wirtualnych gier planszowych. Można ją pobrać tutaj: https://vassalengine.org/download.html

* Pobierz i uruchom moduł **WB-95 - Bzura.vmod**. Jest to podstawowa wersja gry, zawiera jeden scenariusz - Bzura 1939.

## 📁 Struktura repozytorium

``` yaml
Bzura-1939/
│
├── WB-95 - Bzura.vmod      # Podstawowy moduł Vassal – z jednym scenariuszem.
├── resources/              # Wspólne zasoby graficzne: żetony, mapy, okładki, itp.
└── saves/                  # Zawiera zapis z rozstawieniem początkowym:
```
